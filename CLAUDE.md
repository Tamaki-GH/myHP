# myHP — チャットボット付きホームページ プロジェクト

## プロジェクト概要

橘 亜由美（AI エンジニア）のポートフォリオサイトに、チャットボット機能を組み込んだホームページを構築するプロジェクト。

### 構成ファイル

- `index.html` — メインページ（プロフィールカード）
- `serve.rb` — ローカル開発サーバー（WEBrick、ポート 3000）

---

## 開発ルール

### 言語

- **返答はすべて日本語で行う**
- コミットメッセージ・コードコメント・ドキュメントはすべて日本語で書く

### デザイン方針

- **モダンで洗練されたデザイン**を維持する
  - カラーパレット：`--bg: #0f0f1a`、アクセント紫 `#8b5cf6`、シアン `#22d3ee`、ピンク `#f472b6`
  - ダークテーマを基本とする
  - グラスモーフィズム（`backdrop-filter: blur`）・グラデーション・光彩エフェクトを積極活用する
  - アニメーションは控えめに、UX を損なわない範囲で使用する

### レスポンシブ対応

- **スマートフォン・タブレット・デスクトップすべてで見やすいレイアウト**を必ず実装する
- `<meta name="viewport" content="width=device-width, initial-scale=1.0">` を常に含める
- 幅の基準：
  - モバイル：`max-width: 480px`
  - タブレット：`max-width: 768px`
  - デスクトップ：それ以上
- `min()` / `clamp()` などのモダン CSS 関数を活用してフルードレイアウトにする
- タップターゲットは最低 44px 以上を確保する

### コーディング規約

- **コードのコメントは日本語で書く**
- HTML・CSS・JavaScript いずれも同様
- CSS カスタムプロパティ（`--変数名`）でカラー・サイズを管理する
- セマンティックな HTML タグを使用する（`<section>`, `<article>`, `<nav>` など）
- JavaScript はモダン構文（ES2020+）を使用する

### チャットボット実装方針

- チャットボット UI はページ右下に固定フローティングボタンで表示する
- チャットウィンドウはモーダル or スライドイン形式で開く
- スマホでも快適に操作できるよう、入力欄・送信ボタンは十分な大きさを確保する
- ローカル開発中は API キーを HTML に直書きせず、環境変数または外部設定ファイルで管理する

---

## ローカル開発サーバーの起動

```bash
ruby serve.rb
# → http://localhost:3000 で確認
```
